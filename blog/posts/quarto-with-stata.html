仕事でStataを使う必要があったのでメモ

参考にしたページ

  • Stataをモダンな環境で使う https://zenn.dev/nicetak/articles/stata-modern-2025
  • nbstata https://hugetim.github.io/nbstata/

環境構築

  • pythonを使うので、事前にvenvなどで仮想環境をつくっておく
  • 以下のコマンドで必要なパッケージとカーネルをインストールする
    • 公式サイトを見るとpip install nbstataだけになっているが、自分の環境では他のも必要だった
pip install nbstata pyyaml nbformat nbclient
python -m nbstata.install

初期設定

  • jupyterのマジックコマンドのような形で、いくつか初期設定が可能
  • ここではグラフの画像形式をsvgにしておく
  • 詳しくは
    • https://hugetim.github.io/nbstata/user_guide.html#configuration-optional
    • https://hugetim.github.io/nbstata/user_guide.html#set-set
%set graph_format = svg
graph_format was png, is now svg

グラフを書いてみる

  • 都市人口率と高齢化率を計算
gen pop65_prop = pop65p / pop
gen popurban_prop = popurban / pop
  • 散布図を表示
scatter pop65_prop popurban_prop
